https://gologin.com/download/

Step 1: Download and install Gologin Browser

Step 2: Create a free gologin account (3 profiles free for lifetime)

Step 3: Delete all existing profiles

Step 4: Create Profile with that features need to chanage:
        Setting Advanced
          01. Canvas (Noise)
          02. Client Rects (Noise)
          03. WebGL Image (Noise)
          04. Active session lock (Turn On)
          05. Enable Google services (Turn On)

        Create Profile (You can rename profile)

        Open Setting Advanced Again
          01. Enable IndexedDB (Turn On)
        Save



Step 5 : Generate API Key & Profile ID

Step 6: Run Profile login one https://www.linkedin.com and generate search link close browser
 
Step 7: Run the software
